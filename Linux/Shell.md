#### Les types de shell dans [[Linux]]
- **[[sh]]** : Bourne Shell
- **[[bash]]** : Bourne Again Shell
	- `var` est une commande qui peut être appelé n'importe quand.
		- `var=$[nom](commande)` 
- **[[csh]]** : C Shell
- **[[tcsh]]** : Tenex C Shell
- **[[ksh]]** : Korn Shell
- **[[zsh]]** : Zero Shell
